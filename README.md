📍 AtauniLost - Kampüs Kayıp Eşya Portalı AtauniLost, üniversite kampüsü içerisinde kaybolan veya bulunan eşyaların hızlıca sahiplerine ulaştırılmasını sağlayan dinamik bir web platformudur. Atatürk Üniversitesi Bilgisayar Mühendisliği eğitimi kapsamında geliştirilmiştir.

🚀 Temel Özellikler Tam Yetkili Admin Paneli: İlanların onaylanması, düzenlenmesi ve yönetimi için merkezi bir kontrol sistemi.

Kişiselleştirilmiş Kullanıcı Profili: Kullanıcıların kendi ilanlarını takip edebildiği ve kişisel bilgilerini yönetebildiği özel "Profilim" sayfası.

Akıllı Yönlendirme: Admin ve standart kullanıcılar için e-posta bazlı otomatik giriş ve sayfa yönlendirme sistemi.

İnteraktif İlan Yönetimi: Kategori bazlı ilan arama, harita üzerinden konum bilgisi ve yorum sistemi.

🛠️ Kullanılan Teknolojiler Backend: ASP.NET Core MVC

Frontend: HTML5, CSS3, Bootstrap 5 & Bootstrap Icons

Database: Microsoft SQL Server & Entity Framework Core

Language: C#
